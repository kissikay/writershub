# WritersHub 


## Description:
**An interactive app for reading and writing articles**


## ■ Features

- **Reading:**
Normal Users can read the various articles and books that are written by author, for example, Poems, Novels, etc.

- **Writing:**
Writers can upload blogs in any of the available categories

- **Profile Build:**
- Users can buid their profile by reading more books and finding themselves on the readers leader board
- Writers can also build their profiles by getting more reads, followers and also writing more books.

## ■ Activities
- **Create Account or Log into an existing one:**
Both writers and readers create an account or log into their existing accounts 
- **Create Post (Writers only):**
Authors or writers can write or upload their books and publish them.
- **Read books:** Both users and authors can access books from the various categories provided
- **Edit Profile:** All users can edit their profiles (i.e Picture, Username, Bio, etc.)


## ■ Tech Stack(Devs only)
- **Frontend:** HTML,CSS,JS
- **Backend:** ExpressJS,MongoDB
